# Chuck's Creator!

A little web app built from the **"Chuck's Creator!"** Freeform board
(Fun Time Studios LLC). Make characters and scenes by adding things from
**Chuck's Store**, spend **ChuckBux (CB)**, and **"Give it to Chuck!"** for
approval.

## How to play
1. Tap items in the store (right side / bottom on phones) to add them to the board.
2. Drag pieces to move them. Tap a piece to select it, then make it **Bigger /
   Smaller**, send it to the **Front**, or **Delete** it.
3. Pick a **Background** to set the scene.
4. When you're happy, press **🎁 Give it to Chuck!** — Chuck approves your
   work and rewards you with **ChuckBux** and a **★ Prize**.

## The rules (from the board)
- **CB = ChuckBux.** `CB-0` means free; the number is how many ChuckBux it costs.
- **★ PRIZE** items can't be bought with ChuckBux — you need a **Prize**, which
  you earn each time Chuck approves a creation.
- The store only sells when the **sign says OPEN!** Flip it closed/open with the
  button in the header.
- You start with **25 ChuckBux**.

## Running it
It's plain HTML/CSS/JS with no build step or dependencies, and all the art is
inline SVG so it works offline. Just open `index.html` in a browser, or visit
`/chucks-creator/` on the deployed site. Your board and wallet are saved in the
browser (localStorage); **Start Over** clears everything.
